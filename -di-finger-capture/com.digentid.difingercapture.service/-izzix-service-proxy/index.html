IzzixServiceProxy

class IzzixServiceProxy

IzzixRemoteService(:izzix 프로세스)와 AIDL 통신하는 프록시.

기존 IzzixService.instance.method() 호출을 IzzixServiceProxy.instance.method()로 교체하면 자동으로 별도 프로세스에서 JNI 실행.

  • 서비스 연결 대기 (최대 5초)

  • 서비스 프로세스 크래시 시 자동 재연결

  • RemoteException 발생 시 1회 재시도

  • drawMinutiaeOverlay는 IPC 없이 로컬 실행 (순수 Kotlin)
