UseSharedPreference

class UseSharedPreference(context: Context)

DiFingerCapture SDK 촬영 결과를 SharedPreferences에 저장/읽기하는 유틸리티.

com.digentid.difingercapture.activity.FingerActivity 촬영 완료 후 RESULT_OK 수신 시 이 클래스를 통해 특징점 데이터를 읽어옵니다.

사용 예시

val launcher = registerForActivityResult(StartActivityForResult()) { result ->
    if (result.resultCode == RESULT_OK) {
        val pref = UseSharedPreference(context)
        val rounds = pref.getLocalFingerprintARounds()  // 카드A 3라운드
        val bResult = pref.getLocalFingerprintB()       // 카드B 1라운드
    }
}
